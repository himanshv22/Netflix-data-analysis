# Netflix Data Analysis and Visualization

This project performs exploratory data analysis on Netflix datasets using Python.

## Features Implemented
- Data cleaning and preprocessing
- Movies vs TV Shows analysis
- Country-wise content analysis
- Ratings distribution analysis
- Release year trend visualization
- Genre analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

The objective of this project is to analyze Netflix content trends and visualize insights from real-world entertainment datasets.
